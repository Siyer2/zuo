== Synopsis
[verse]
zuo [-h|--help] [-v|--version] <subcommand> [<subcommand-options>...] [<subcommand-arguments>...]

== Description

Zuo is an i3-like tiling window manager for macOS

*zuo* command line program is used to manipulate Zuo and query its state.

See https://nikitabobko.github.io/Zuo/commands for available <subcommand> options

See each <subcommand> individual man page for <subcommand-options> and <subcommand-arguments>

== Options

-h, --help:: Print help
